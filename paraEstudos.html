Tags básicas

(p) paragrafo

(a) tag de link, junto com href

(header) define um cabeçalho

(section) define uma seção

(article) define um artigo

Clique aqui!

(div) define uma divisao

(footer) define um rodapé

(nav) define uma area de navegação (como menus)

(table) define uma tabela

(ol) define uma lista ordenada

(ul) define uma lista nao ordenada

(li) define o item de uma lista

(form) define um formulario

(input) define os campos do formulario

(textarea) define uma area para o usuario digitar um texto

(button) define um botao

(img) permite uma imagem junto com src
